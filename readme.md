## `⚡` Descripción

> **Cat's Bot** es un bot **multifuncional** desarrollado exclusivamente para la comunidad de **Las Tortuguitas de Ezku**.
> Integra moderación, música, entretenimiento y utilidades en un solo sistema estable y disponible las 24 horas.

<br/>

<img src="https://capsule-render.vercel.app/api?type=rect&color=7c3aed,ec4899&height=1" width="100%"/>

<br/>

## `🧩` Módulos del Sistema

<div align="center">

| &nbsp; | Módulo | Descripción |
|:---:|:---|:---|
| 🛡️ | **Moderación** | Ban, kick, mute, warn y gestión de roles |
| 🎵 | **Música** | Reproductor con cola, pausa y controles |
| 🎉 | **Bienvenidas** | Mensajes automáticos para nuevos miembros |
| 🎮 | **Entretenimiento** | Trivia, memes, juegos y comandos sociales |
| 📊 | **Estadísticas** | Perfiles de usuario e info del servidor |
| ⚙️ | **Utilidades** | Polls, recordatorios y herramientas generales |

</div>
